# Cisco DHCP Snooping Lab

## 📌 Project Overview
This project demonstrates DHCP Snooping configuration in Cisco Packet Tracer.

## 🛠 Technologies Used
- Cisco 2960 Switch
- Cisco 2811 Router
- DHCP Server
- DHCP Snooping

## 🔐 Security Features
- Trusted and Untrusted ports configuration
- DHCP Snooping enabled globally
- Protection against DHCP Spoofing attacks
